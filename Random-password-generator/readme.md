<h2>âš™ï¸ How It Works</h2>

<h3>1. Slider for Password Length</h3>
<p>The password length is controlled by a <code>&lt;input type="range"&gt;</code> element. The current value is displayed dynamically beside the slider.</p>

<h3>2. Character Selection</h3>
<p>The user can select which types of characters they want in the password using checkboxes:</p>
<ul>
  <li>Lowercase letters</li>
  <li>Uppercase letters</li>
  <li>Numbers</li>
  <li>Symbols</li>
</ul>
<p>If no checkbox is selected, the user is alerted to select at least one character type.</p>

<h3>3. Password Generation</h3>
<p>The password is generated by randomly picking characters from the selected sets. The length of the password matches the slider value.</p>

<h3>4. Copy to Clipboard</h3>
<p>Clicking the <strong>copy icon</strong> copies the generated password to the clipboard. The icon temporarily changes to a checkmark to provide visual feedback.</p>
